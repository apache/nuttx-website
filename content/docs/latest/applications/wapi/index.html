WAPI (Wireless API)ÔÉÅ

WAPI (Wireless API) provides an easy-to-use function set to configure wireless network interfaces on a GNU/Linux system, originally.

One can provide use wapi to connect to Access Points, configure an Access Point, set Wi-Fi related configurations and get information about it.

Currently, wapi is the only interface provided to handle Wi-Fi networking from the userspace.
