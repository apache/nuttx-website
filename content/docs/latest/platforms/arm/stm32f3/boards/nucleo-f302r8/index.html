ST Nucleo F302R8

The Nucleo F302R8 is a member of the Nucleo-64 board family.

Configurations

ihm07m1_f32 and ihm07m1_b16:

These examples are dedicated for the X-NUCLEO-IHM07M1 expansion board based on L6230 DMOS driver for three-phase brushless DC motors.

X-NUCLEO-IHM07M1 must be configured to work with FOC and 3-shunt resistors. See ST documentation for details.
